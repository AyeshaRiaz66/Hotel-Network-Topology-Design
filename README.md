# 🌐 Hotel Management Network System

A simulated hotel communication network designed using Cisco Packet Tracer to enable efficient data transmission between hotel rooms and management systems.

🚀 Features
- Network topology design for hotel infrastructure
- Router and switch configuration
- IP addressing scheme implementation
- Packet transmission simulation
- Communication between rooms and central management

🛠️ Technologies Used
- Cisco Packet Tracer
- Routing & Switching
- IP Addressing
- Network Configuration

📚 Concepts Applied
- Network Topology Design
- Router & Switch Configuration
- Subnetting
- Routing Protocols
- Packet Transmission Simulation

🎯 Objective
To design and simulate a functional hotel communication network demonstrating practical networking and routing concepts.
